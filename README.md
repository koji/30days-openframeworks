Drawing Primitives

#### Line
```cpp
ofSetLineWidth(5);
ofDrawLine(x1, y1, x2, y2);
```

```java
strokeWeight(5);
line(x1, y1, x2, y2);
```

#### Rect
```cpp
ofRect(x, y, w, h);
```

```java
rect(x, y, w, h);
```

#### Triangle
```cpp
ofTriangle(x1, y1, x2, y2, x3, y3);
```

```java
triangle(x1, y1, x2, y2, x3, y3);
```

#### Circle
```cpp
ofDrawCircle(x, y, r);
```

```java
ellipse(x, y, r);
```

#### fill & nofill
```cpp
ofFill(r, g, b, alpha);
ofNoFill();
```

```java
fill(r, g, b, alpha);
noFill();
```

#### dot
```cpp
ofPoint(x, y);
```

```java
dot(x, y);
```
